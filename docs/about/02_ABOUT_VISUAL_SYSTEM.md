# Introduction

This document defines how the About section should look.

Not how it should be coded.

Not how it should be animated.

Only how visitors should visually experience Chapter Two.

The visual language must continue seamlessly from the Hero.

Visitors should never feel they entered another website.

---

# Visual Objective

The Hero builds trust.

The About section builds emotional connection.

The visual experience should quietly answer:

"This organization genuinely works with people."

Without explicitly saying it.

Desktop

Two-column layout.

Left

Large documentary image.

Right

Editorial content.

Quote.

Supporting text.

Tablet

Balanced two-column layout.

Mobile

Photography first.

Story second.

Preserve the emotional sequence.

---

# Photography System

Primary Image

One dominant image.

Large.

Emotional.

Authentic.

Supporting Images

Three to five smaller documentary images.

Natural composition.

Different moments.

Different perspectives.

---

# Preferred Photography

Community mentoring.

UMKM activities.

Government collaboration.

Creative workshops.

Student mentoring.

Innovation programs.

Real conversations.

Real environments.

---

# Image Treatment

Large radius.

Natural lighting.

Soft shadows.

No heavy editing.

No artificial saturation.

No dramatic filters.

Photography should feel timeless.

---

# Typography

Label

Small.

Uppercase.

Brand accent.

Wide letter spacing.

Headline

Large.

Editorial.

Maximum three lines.

Description

Comfortable reading width.

Approximately 60–70 characters per line.

Quote

Elegant.

Minimal.

Supporting element.

Never dominate the layout.

---

# White Space

Large vertical spacing.

Comfortable margins.

Premium breathing room.

Empty space is intentional.

Never compress content.

---

# Background

Continue Hero background.

Soft white.

Very subtle gradient.

Light radial glow.

Minimal texture.

Invisible grid only if necessary.

Visitors should focus on the story.

---

# Color Strategy

Primary

White

Off White

Light Gray

Charcoal

Brand Maroon

Accent colors should be extremely limited.

---

# Shape Language

Rounded corners.

Soft cards.

Thin borders.

Subtle depth.

Nothing aggressive.

# Visual Balance

Neither side should dominate.
