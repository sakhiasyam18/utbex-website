# OBJECTIVE

Transform the website into a documentary experience.

Do NOT create a portfolio.

Do NOT create a gallery.

Do NOT create a marketing landing page.

Create a visual journey.

---

# EXISTING PHOTOGRAPHY

The project already contains authentic documentary photographs.

Use them.

Study them.

Assign each image an intentional role.

Never place images randomly.

Never repeat similar emotional moments.

Every photograph must contribute to the story.

---

# STORYTELLING RULES

Every chapter should answer only one question.

Hero

Who are we?

↓

About

Why do we exist?

↓

Portfolio

What have we accomplished?

↓

Impact

What changed?

↓

Footer

Where do we continue from here?

Maintain this narrative.

---

# PHOTOGRAPHY RULES

Prioritize:

Real people.

Real interaction.

Real communities.

Real implementation.

Real achievements.

Avoid decorative imagery.

Avoid unnecessary repetition.

One dominant image.

Several supporting images.

Many contextual details.

---

# COMPOSITION RULES

Alternate layouts.

Alternate image scales.

Alternate reading rhythm.

Alternate emotional intensity.

Avoid repeating identical visual structures.

The website should feel handcrafted,

not generated.

---

# PANORAMIC IMAGES

Treat panoramic documentary photographs as chapter dividers.

Allow them to breathe.

Never reduce them to gallery cards.

Never crop them aggressively.

---

government partnerships,

billboards,

Recognition confirms the journey.

It never begins the journey.
