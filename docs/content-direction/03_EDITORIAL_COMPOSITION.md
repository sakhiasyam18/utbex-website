# Introduction

The website should never resemble a collection of UI components.

It should resemble a carefully designed editorial publication.

Visitors should feel that every photograph,

every paragraph,

every block,

and every empty space has been intentionally composed.

The composition itself tells the story.

---

# Philosophy

Editorial composition is not symmetry.

Editorial composition is controlled imbalance.

Some moments deserve attention.

Some moments deserve silence.

Some moments deserve a single breathtaking photograph.

The composition controls emotion before words do.

---

# Editorial Flow

Observe

↓

Pause

↓

Read

↓

Observe

↓

Continue

Every chapter should alternate between visual intensity and visual silence.

Avoid maintaining the same rhythm for too long.

---

# Composition Hierarchy

Hero Composition

↓

Feature Composition

↓

Supporting Composition

↓

Gallery Composition

↓

Closing Composition

Each composition has a different emotional responsibility.

---

# Hero Composition

One dominant photograph.

Minimal typography.

Large breathing space.

Visitors should instantly understand:

"This story matters."

Never overcrowd the Hero.

---

# Feature Composition

Feature compositions introduce the most important moments.

Examples:

Major workshops.

Community engagement.

Public presentations.

Government collaboration.

Use:

Large imagery.

Editorial typography.

Generous whitespace.

The photograph remains dominant.

---

# Split Composition

Recommended layout:

Large Image

↓

Editorial Text

or

Editorial Text

↓

Large Image

Never repeat the same left-right pattern.

Alternate the visual direction.

This creates reading rhythm.

---

# Mosaic Composition

Mosaic layouts should appear only once or twice.

Purpose:

Show activity diversity.

Not every project deserves a mosaic.

Reserve this layout for chapters rich in visual documentation.

---

# Documentary Strip

A sequence of narrow documentary photographs.

Purpose:

Show progression.

Examples:

Preparation

↓

Training

↓

Discussion

↓

Implementation

↓

Celebration

Visitors should experience movement through time.

---

# Cinematic Composition

Use one panoramic image across nearly the full width of the page.

This creates an emotional pause.

Ideal positions:

Between About and Portfolio.

Between Portfolio and Impact.

Never use panoramic photographs inside cards.

They deserve their own visual chapter.

---

# Editorial Quote

Some chapters may end with:

One meaningful sentence.

One testimonial.

One observation.

Surrounded by generous whitespace.

The quote allows visitors to breathe before continuing.

---

---

# Visual Weight

One dominant visual.

One supporting visual.

Several contextual visuals.

Many detail visuals.

Visitors should always know where to look first.

---

# Reading Direction

Composition should naturally guide the eye.

Photography.

↓

Headline.

↓

Body.

↓

Supporting image.

↓

Next section.

Never force visitors to search for the story.

---

# Negative Space

Whitespace is a design element.

Do not fill every available area.

Empty space creates:

Elegance.

Confidence.

Editorial quality.

Silence is part of storytelling.

---

# Image Cropping

Protect:

Faces.

Eye contact.

Hands.

Interaction.

Context.

Never crop purely for symmetry.

Always preserve narrative.

---

# Documentary Sequence

Recommended visual sequence:

Wide Environment

↓

Human Activity

↓

Close Detail

↓

Interaction

↓

Result

↓

Celebration

↓

Reflection

Visitors should feel progression.

---

# Alternating Layout

Avoid repeating identical layouts.

Instead alternate:

Image Left

↓

Image Right

↓

Centered

↓

Panorama

↓

Editorial Quote

↓

Gallery

↓

Feature Image

This creates visual rhythm.

---

# Component Relationship

Components should disappear behind the editorial composition.

Cards.

Containers.

Glass.

Borders.

Everything exists only to support storytelling.

---

# Photography Priority

Never allow decorative UI to compete with documentary photography.

Real moments always receive the highest visual importance.

---

# Responsive Behaviour

Desktop

Editorial layouts remain expressive.

Tablet

Preserve hierarchy.

Mobile

Stack naturally.

Maintain emotional pacing,

not desktop symmetry.

---

# Accessibility

Editorial compositions should remain understandable without relying solely on visuals.

Typography,

image order,

and semantic structure must work together.

---

# Editorial Composition Checklist

✓ Documentary.

✓ Human.

✓ Cinematic.

✓ Premium.

✓ Asymmetrical.

✓ Story-driven.

✓ Calm.

✓ Memorable.

---

# Final Principle

Visitors should never think,

"This layout is beautiful."

They should simply continue scrolling because the story naturally pulls them forward.

That is the purpose of Editorial Composition.
