# Photography Rules

Arrange photography intentionally.

Do not display photographs randomly.

Create progression.

Beginning.

↓

Preparation.

↓

Learning.

↓

Implementation.

↓

Community.

↓

Achievement.

↓

Reflection.

---

# Hero Photography

Use only one emotionally dominant image.

It should immediately communicate:

Purpose.

People.

Mission.

Never overload the Hero with multiple competing photographs.

---

# Editorial Composition

Avoid repetitive layouts.

Alternate:

Feature Image

↓

Split Layout

↓

Panoramic Story

↓

Editorial Layout

↓

Gallery

↓

Feature Image

The page should feel like a premium magazine,

not a template.

---

# Panoramic Photography

The project already includes wide-format documentary photographs.

Treat these as chapter separators.

Never reduce them into ordinary cards.

Allow them to span nearly the entire viewport.

These images create emotional breathing space.

---

# Image Hierarchy

Only one image leads.

Every other image supports.

Avoid displaying many equally important images together.

Visitors should instantly know where to look first.

---

↓

↓
