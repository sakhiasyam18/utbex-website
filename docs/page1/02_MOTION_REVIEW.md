# 02_MOTION_REVIEW.md

# UTBEX DIGITAL EXPERIENCE SYSTEM

## MOTION REVIEW

Status:
Review Phase

Priority:
HIGH

Audience:

Frontend Engineer

Creative Developer

---

# Objective

The UI and Responsive phases have been approved.

This phase focuses on refining interaction through subtle, purposeful motion.

Motion should support storytelling, never distract from it.

---

# Motion Philosophy

Motion is invisible.

Users should remember the story, not the animation.

Every animation must have a narrative purpose.

---

# Motion Principles

- Calm
- Elegant
- Premium
- Documentary
- Natural
- Smooth

Avoid playful or exaggerated effects.

---

# Global Motion

Verify:

- Smooth page loading.
- Smooth scrolling.
- No layout jump.
- No animation stutter.

---

# Hero

Review:

- Headline reveal.
- CTA fade.
- Hero cards entrance.
- Background glow.

Animation should feel welcoming, not dramatic.

---

# Manifesto

Review:

- Typography reveal.
- Scroll opacity.
- Gentle fade.

No aggressive movement.

---

# Human Story

Review:

- Image reveal.
- Text reveal.
- Split layout transition.

Focus attention on the human story.

---

# Portfolio

Review:

- Card reveal.
- Stagger timing.
- Hover interaction.

Keep motion subtle.

---

# Full Width Images

Review:

- Parallax (very light).
- Fade between chapters.

Avoid excessive movement.

---

# Impact

Review:

- Counter animation.
- Statistic reveal.

Fast and clean.

No bouncing numbers.

---

# CTA

Review:

- Button hover.
- Card appearance.
- Background transition.

Create a warm invitation.

---

# Navigation

Review:

- Navbar transparency.
- Active link.
- Mobile menu.

Navigation should feel lightweight.

---

# Hover States

Verify:

- Buttons
- Cards
- Links
- Images

Hover should communicate interactivity.

Never become decorative.

---

# Scroll Experience

Verify:

- Every chapter enters naturally.
- No simultaneous animations.
- Comfortable pacing.

Scrolling should feel cinematic.

---

# Accessibility

Respect:

- prefers-reduced-motion
- Keyboard navigation
- Focus visibility

Motion must never reduce usability.

---

# Performance

Target:

- 60 FPS
- No animation jank
- GPU-friendly transforms
- Opacity & Transform only

Avoid animating:

- width
- height
- top
- left

Prefer:

- opacity
- transform

---

# Deliverables

Review every animation.

Document:

- Current behavior
- Suggested improvements
- Performance impact

Implement after approval.

---

# Definition of Done

✓ Motion supports storytelling.

✓ Motion remains subtle.

✓ Motion feels premium.

✓ Scrolling feels effortless.

✓ No distracting animations.

✓ Performance remains smooth.

---

# Final Principle

Good motion is rarely noticed.

Great motion is quietly remembered through the feeling it creates.

End of Document.
