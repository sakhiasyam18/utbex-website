# UTBEX DIGITAL EXPERIENCE SYSTEM

## MOTION SYSTEM

Version: 1.0

Status: LOCKED

Language: English

Audience:

Creative Director

Motion Designer

Frontend Engineer

AI Coding Assistant

---

# Introduction

Motion is not decoration.

Motion is communication.

Every movement should improve orientation,

guide attention,

or reinforce emotion.

If an animation serves no purpose,

remove it.

---

# Motion Philosophy

The website should never feel animated.

It should feel alive.

Visitors should experience calm,

continuous,

natural movement.

Nothing should feel mechanical.

Nothing should feel exaggerated.

---

# Emotional Direction

Calm.

Elegant.

Premium.

Confident.

Human.

Timeless.

Motion should never feel playful.

Never theatrical.

---

# Motion Hierarchy

Page Transition

↓

Section Reveal

↓

Component Reveal

↓

Interaction

↓

Ambient Motion

↓

Micro Feedback

The closer an animation is to the user,

the shorter it becomes.

---

# Page Reveal

Soft opacity.

Small translate.

Comfortable easing.

No dramatic entrances.

The page should appear naturally,

like turning the first page of a book.

---

# Section Reveal

Each section enters independently.

Use:

• opacity

• translateY

• slight stagger

Avoid:

• zoom

• rotation

• bounce

• flip

---

# Scroll Motion

Scrolling should reveal content,

not animate it.

Movement should follow the user's intention.

Never compete with scrolling.

---

# Hover Motion

Hover should confirm interaction.

Recommended:

• tiny lift

• opacity refinement

• subtle underline

• soft shadow refinement

Never surprise users.

---

# Glass Motion

Glass should remain stable.

Never pulse.

Never shimmer.

Never continuously animate blur.

Glass should feel physically present.

---

# Ambient Motion

Very slow movement is allowed.

Examples:

• slow radial light drift

• soft background diffusion

• gentle atmospheric movement

Duration should be measured in several seconds.

Visitors should barely perceive it.

---

# Lighting Motion

Lighting should breathe,

not flicker.

Opacity changes only.

Very slow transitions.

Never create visible moving lights.

---

# Navigation Motion

Navigation glides.

It never jumps.

Reveal.

Hide.

Scroll.

Hover.

Everything should feel effortless.

---

# Image Motion

Images remain stable.

Very small parallax is allowed.

Maximum movement should remain subtle.

Photography must always stay the focus.

---

# Performance

Animate only:

Transform.

Opacity.

Filter (carefully).

Avoid layout recalculation.

Avoid expensive paint operations.

Maintain smooth rendering.

---

# Accessibility

Respect prefers-reduced-motion.

Disable:

Ambient movement.

Parallax.

Continuous animation.

Keep:

Opacity.

Instant feedback.

Accessibility always overrides aesthetics.

---

# Motion Checklist

✓ Calm.

✓ Smooth.

✓ Lightweight.

✓ Purposeful.

✓ Premium.

✓ Consistent.

✓ Accessible.

✓ Invisible.

---

# Final Principle

Visitors should never remember the animations.

They should simply remember that everything felt effortless.

That is the purpose of the UTBEX Motion System.
