# Introduction

Not every photograph deserves the same visual importance.

Hierarchy determines where visitors look first,

how long they stay,

and what they remember.

Image hierarchy is one of the strongest storytelling tools in the entire website.

---

# Philosophy

The visitor should never feel like they are browsing a gallery.

Instead,

they should feel like every image appears exactly when the story needs it.

Large moments.

Quiet moments.

Evidence.

Reflection.

The rhythm should feel intentional.

---

Every chapter should follow this hierarchy.

---

# Priority A

Hero Image

Purpose:

Capture emotion immediately.

Only one image should dominate the screen.

Characteristics:

• strongest composition

• strongest lighting

• strongest emotional impact

• authentic people

• clean background

Large.

Confident.

Unforgettable.

---

# Priority B

Editorial Image

Purpose:

Continue the narrative.

Usually occupies half or two-thirds of the layout.

These images carry the story forward.

Examples:

Workshop.

Presentation.

Community discussion.

Field activity.

---

# Priority C

Supporting Image

Purpose:

Support the editorial image.

Provide additional perspective.

Never compete with the primary image.

---

# Priority D

Context Image

Purpose:

Explain location.

Architecture.

Village.

Campus.

Training room.

Audience.

These images create atmosphere.

---

# Priority E

Detail Image

Purpose:

Reveal realism.

Examples:

Hands writing.

Laptop screen.

Training material.

Equipment.

Signage.

Micro interaction.

These images increase credibility.

---

# Priority F

Closing Image

Purpose:

Emotionally conclude a chapter.

Examples:

Group photo.

Celebration.

Award.

Community smile.

Handshake.

Sunset.

Reflection.

These images should encourage visitors to continue scrolling.

---

# Full Width Documentary Image

The project already contains several panoramic and wide-format documentary photographs.

These images must never become ordinary gallery items.

Instead,

treat them as chapter dividers.

Example:

Hero

↓

About

↓

Full Width Documentary Image

↓

Portfolio

↓

Full Width Documentary Image

↓

Impact

↓

Footer

The image becomes a breathing moment.

Not decoration.

---

# Hero Image Rules

Never reuse Hero-quality photographs later on the page.

Each Hero image should remain unique.

It establishes emotional identity.

# Image Scale

Large

Creates emotion.

Medium

Provides information.

Small

Provides detail.

Micro

Provides authenticity.

Never display every image at the same size.

Variation creates editorial rhythm.

---

# Visual Weight

Not every image deserves equal attention.

One dominant image.

Two supporting images.

Several contextual images.

Many detail images.

Maintain this visual hierarchy consistently.

---

# White Space

Large images deserve generous breathing room.

Do not place images too close together.

Allow the visitor to pause.

Silence strengthens photography.

---

# Relationship With Text

Images lead.

Text follows.

Never force visitors to read before seeing.

Photography attracts.

Typography explains.

---

# Relationship With Components

Cards should adapt to photography.

Photography should never adapt to card limitations.

The story always comes first.

---

# Gallery Rules

Avoid:

Equal-size grids.

Symmetrical galleries.

Uniform cards.

Perfect repetition.

Instead create:

Editorial compositions.

Asymmetrical balance.

Intentional rhythm.

Visual breathing space.

---

# Panorama Usage

Panoramic photographs should appear only at major emotional transitions.

Never inside small cards.

Never cropped aggressively.

Allow them to breathe across the screen.

They create cinematic pauses.

---

# Portrait Photography

Portrait orientation should highlight:

People.

Faces.

Interactions.

Learning moments.

Portraits should slow the reading pace.

---

# Landscape Photography

Landscape orientation should establish:

Environment.

Scale.

Community.

Projects.

Context.

Landscape expands the story.

---

# Achievement Photography

Recognition should appear near the end of a narrative.

Examples:

Awards.

Government collaboration.

Billboards.

Certificates.

Media coverage.

Recognition validates trust.

It should never become the opening chapter.

---

# Group Photography

Avoid placing group photos at the beginning.

Group photographs conclude stories.

They communicate:

Completion.

Success.

Belonging.

Celebration.

---

# Responsive Behaviour

Desktop

Editorial hierarchy.

Tablet

Preserve hierarchy.

Mobile

Preserve emotional importance,

even if image sizes change.

Never flatten the hierarchy.

---

# Accessibility

Every image must communicate meaning.

Alt text should describe:

People.

Activity.

Context.

Purpose.

Never use decorative alt text.

---

# Image Hierarchy Checklist

✓ One dominant image.

✓ Documentary rhythm.

✓ Editorial composition.

✓ Human-centered.

✓ Authentic.

✓ Balanced.

✓ Cinematic.

✓ Story-driven.

---

# Final Principle

Visitors should never think,

"These images are arranged beautifully."

Instead,

they should feel that every photograph appeared at exactly the right moment in the story.

That is the purpose of Image Hierarchy.
