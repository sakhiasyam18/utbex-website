# Introduction

The Navigation is the only interface element that accompanies visitors throughout the entire website.

Its visual language must remain calm,

consistent,

and elegant.

Unlike Hero,

it should never become the visual focus.

Instead,

it quietly elevates every section beneath it.

---

# Visual Goal

Create the feeling of a floating editorial navigation.

Not a browser bar.

Not a dashboard.

Not a SaaS navbar.

A premium navigation suspended above the storytelling.

---

# Visual DNA

Lightweight.

Floating.

Minimal.

Editorial.

Modern.

Alive.

Every visual detail should communicate refinement.

---

# Composition

Floating Container

↓

Logo

↓

Primary Navigation

↓

Primary CTA

Nothing else.

Avoid secondary menus unless absolutely necessary.

---

# Floating Container

The Navigation should never touch the browser edges.

Leave generous horizontal margins.

Large border radius.

Floating appearance.

Comfortable breathing space.

---

# Glass Surface

Required layers:

• Semi-transparent white surface

• Soft backdrop blur

• Thin translucent border

• Ambient shadow

• Extremely soft top highlight

The surface should feel like polished glass.

---

# Ambient Background

To avoid a flat white appearance,

the Navigation may include:

• Soft radial light

• Warm blur

• Gentle maroon ambient glow

• Very subtle neutral gradient

Opacity should remain below 5%.

Visitors should feel depth,

not see the effect.

---

# Lighting System

Introduce invisible lighting.

Examples:

• Soft top-left glow

• Small bottom-right reflection

• Gentle center diffusion

Never use visible neon strips.

Never create cyberpunk lighting.

Light exists only to create atmosphere.

---

# Shadow System

Large blur radius.

Low opacity.

Wide spread.

Natural elevation.

Avoid harsh drop shadows.

The Navigation should appear softly lifted.

---

# Border System

One thin translucent border.

Approximately 1px.

Opacity between 10–20%.

No double borders.

No thick outlines.

---

# Color Distribution

90%

White

Gray

Neutral

10%

UTBEX Accent

Only for:

Hover

Active

CTA

Never color the entire Navigation.

---

# Typography

Medium weight.

Comfortable tracking.

High readability.

Navigation should remain visually lighter than Hero typography.

---

# CTA Style

Primary CTA should feel integrated.

Rounded capsule.

Soft shadow.

Subtle hover glow.

Never oversized.

Never brighter than Hero CTA.

---

# Hover State

Smooth brightness transition.

Tiny glow.

Thin underline.

Soft elevation.

Hover should reward interaction,

not announce it.

---

# Active State

Choose one indicator only.

Recommended:

• Soft glowing dot

or

• Thin accent underline

or

• Light capsule background

Never combine multiple active indicators.

---

# Blur Strategy

Backdrop blur:

Soft.

Stable.

Readable.

The page behind should remain recognizable,

not disappear.

---

# Responsive Appearance

Desktop

Floating editorial bar.

Tablet

Slightly condensed.

Mobile

Floating capsule with drawer navigation.

Maintain the same visual language.

---

# Visual Checklist

✓ Floating.

✓ Glass.

✓ Soft blur.

✓ Ambient lighting.

✓ Premium shadow.

✓ Calm typography.

✓ Invisible elegance.

✓ Consistent with Hero.

---

# Final Principle

The Navigation should feel like a carefully crafted piece of glass floating above the story.

Visitors should remember the experience,

never the interface.
