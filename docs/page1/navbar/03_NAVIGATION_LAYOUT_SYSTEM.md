# Introduction

Navigation should establish orientation without disrupting immersion.

The layout must remain clean,

predictable,

and balanced across every viewport.

The visitor should always know where they are,

without consciously looking for the navigation.

---

# Layout Philosophy

Navigation floats above the website.

It is never attached to the browser edge.

It feels independent,

light,

and carefully positioned.

Imagine a premium object suspended in space.

---

# Global Structure

Floating Container

↓

Brand Identity

↓

Primary Navigation

↓

Primary CTA

Maintain this hierarchy at every breakpoint.

---

# Container Width

Desktop

Centered.

Maximum width between 1200px–1320px.

Never full width.

Leave generous margins.

Tablet

Slightly reduced width.

Maintain floating appearance.

Mobile

Compact floating capsule.

Never stretch edge-to-edge.

---

# Vertical Position

Navigation should breathe.

Desktop

Top offset approximately 24–36px.

Tablet

20–28px.

Mobile

16–20px.

Never attach directly to the browser edge.

---

# Internal Layout

Logo

↓

Navigation Links

↓

Primary CTA

Balanced using Flexbox.

Navigation should feel mathematically centered.

---

# Logo Placement

Always left.

Maintain generous spacing around the logo.

Logo should remain the visual anchor.

Never center the logo on desktop.

---

# Navigation Links

Always centered.

Equal spacing.

Never compressed.

Comfortable click targets.

Horizontal rhythm should remain consistent.

---

# CTA Placement

Always right.

Separated from navigation.

Enough breathing room to feel independent.

Never oversized.

---

# Floating Spacing

Outside spacing creates elegance.

Inside spacing creates comfort.

Protect both.

Do not sacrifice margins to fit more content.

---

# Scroll State

Navigation keeps the same position.

Only atmosphere changes.

Glass becomes slightly stronger.

Shadow deepens slightly.

Blur increases subtly.

Layout never shifts.

---

# Mobile Layout

Logo

↓

Hamburger Button

Drawer opens from the right.

Use a floating panel.

Never full-screen takeover.

The background remains softly visible through blur.

---

# Drawer Layout

Logo

↓

Navigation Links

↓

CTA

↓

Social Links (optional)

Maintain generous spacing.

No dense lists.

---

# Alignment

Desktop

Horizontal.

Tablet

Horizontal.

Mobile

Vertical inside drawer.

Always maintain one visual rhythm.

---

# Safe Area

Respect browser safe areas.

Respect device notches.

Maintain consistent padding.

Never allow UI to touch screen edges.

---

# Visual Balance

The Navigation should occupy less visual weight than Hero.

Visitors should immediately notice the page,

not the Navigation.

---

# Layout Checklist

✓ Floating.

✓ Balanced.

✓ Comfortable spacing.

✓ Premium proportions.

✓ Responsive.

✓ Stable while scrolling.

✓ Consistent across devices.

---

# Final Principle

The Navigation should feel perfectly positioned,

as if it naturally belongs above every chapter without ever asking for attention.
