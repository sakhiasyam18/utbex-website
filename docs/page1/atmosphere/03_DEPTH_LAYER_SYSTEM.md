# Introduction

Depth is the invisible architecture of the UTBEX experience.

Visitors should never perceive a flat interface.

Instead,

they should feel that every section exists within a carefully crafted spatial environment.

Depth is created by relationships,

not decoration.

---

# Philosophy

Objects never create depth.

Light creates depth.

Blur creates depth.

Spacing creates depth.

Atmosphere creates depth.

Every layer should feel naturally separated,

without obvious visual tricks.

---

---

# Background Layer

The background always remains the quietest element.

It creates space,

not attention.

Never introduce visible graphics.

Only atmosphere.

---

# Content Layer

Typography.

Photography.

Editorial content.

These elements remain the visual focus.

Depth exists only to support readability.

---

# Floating Layer

Floating objects include:

Navigation.

Glass cards.

Floating CTA.

Drawer.

These elements should appear gently suspended above the content,

never detached from it.

---

# Glass Layer

Glass naturally separates foreground from background.

Glass should never create artificial depth.

Instead,

it softly reveals that another layer exists behind it.

---

# Blur Layer

Blur softens transitions between layers.

Blur never hides information.

Blur should always improve spatial perception.

---

# Shadow Layer

Shadow confirms elevation.

Shadow never creates elevation by itself.

Use:

large blur

low opacity

soft diffusion

wide spread

Avoid harsh shadows.

---

# Atmospheric Layer

Depth is reinforced by:

soft radial light

warm diffusion

ambient gradients

subtle reflections

These layers should remain almost invisible.

---

# Visual Distance

Not every layer should have equal contrast.

Higher layers receive:

slightly higher clarity

slightly stronger contrast

slightly sharper edges

Lower layers become softer.

---

# Spatial Rhythm

Hero

Highest spatial depth.

↓

About

Slightly calmer.

↓

Portfolio

Balanced.

↓

Impact

Medium depth.

↓

Footer

Gradually flatten toward the end.

Visitors should unconsciously feel the journey slowing down.

---

# Three-Dimensional Feeling

The website should feel architectural,

not decorative.

Imagine walking through a modern gallery,

not playing inside a 3D application.

---

# Motion Relationship

Depth responds subtly to movement.

Small parallax.

Tiny blur adjustments.

Gentle opacity transitions.

Never obvious perspective shifts.

---

# Responsive Behaviour

Depth remains consistent across all devices.

Reduce scale,

never remove the layering system.

Atmosphere should survive every screen size.

---

# Performance

Depth should rely primarily on:

CSS gradients

Backdrop blur

Opacity

Transform

Avoid expensive rendering techniques.

---

# Depth Checklist

✓ Layered.

✓ Architectural.

✓ Calm.

✓ Invisible.

✓ Premium.

✓ Lightweight.

✓ Editorial.

✓ Human.

---

# Final Principle

Visitors should never think,

"This website has depth."

They should simply feel that every element naturally belongs within the same carefully crafted space.
