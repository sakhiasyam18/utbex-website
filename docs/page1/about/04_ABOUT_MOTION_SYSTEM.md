# Introduction

Motion exists to support storytelling.

Never decoration.

Never distraction.

Visitors should almost not notice the animation.

They should only feel that the experience is alive.

---

# Motion Philosophy

Calm.

Natural.

Elegant.

Human.

Premium.

Purposeful.

Every movement should have intention.
