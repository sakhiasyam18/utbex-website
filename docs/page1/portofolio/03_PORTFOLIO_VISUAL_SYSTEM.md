# Introduction

Portfolio is not visually separated from About.

It is a continuation.

Visitors should never feel they entered a new section.

The atmosphere evolves naturally.

---

# Visual Goal

Create the feeling of exploring a carefully curated documentary exhibition.

Not browsing a website.

Every project deserves attention.

Nothing competes for attention.

# Layout Philosophy

Never build equal cards.

Never align everything perfectly.

Avoid mechanical grids.

Use controlled asymmetry.

Everything feels intentional.

Nothing feels random.

---

# Photography Composition

One dominant image.

Several supporting moments.

Different scales.

Different crop ratios.

Natural overlap.

Soft depth.

Large breathing space.

---

# Typography

Large editorial headline.

Short supporting paragraph.

Compact metadata.

Readable captions.

Quiet labels.

No oversized UI elements.

---

# Documentary Labels

Instead of badges,

use documentary captions.

Examples

Community Workshop

Malang, Indonesia

2025

Small.

Quiet.

Elegant.

---

# Project Card Philosophy

Cards should disappear.

The story should remain.

The interface should never become the focus.

---

# Image Treatment

Rounded corners.

Soft shadows.

Natural colors.

Very light contrast enhancement.

No heavy filters.

No glossy effects.

---

# White Space

Large.

Intentional.

Comfortable.

Allow every project to breathe.

Premium interfaces never feel crowded.

---

# Visual Transition

Background changes slowly.

Lighting becomes warmer.

Typography remains consistent.

Motion becomes slightly calmer.

Portfolio should feel deeper than About.

---

# Responsive Strategy

Desktop

Editorial composition.

Tablet

Balanced storytelling.

Mobile

One story at a time.

Never overwhelm the visitor.

---

# Success Metrics

✓ Documentary feeling.

✓ Premium composition.

✓ Human-centered visuals.

✓ No gallery feeling.

✓ No template feeling.

✓ Seamless continuation from About.

---

# Final Principle

The Portfolio should not look designed.

It should look carefully curated.

Like an exhibition where every project has earned its place.
