---

# Introduction

Lighting is not decoration.

Lighting creates atmosphere.

Visitors should never consciously notice the lighting.

They should only feel that Portfolio has a different emotional temperature from Hero and About.

---

# Emotional Direction

Hero

Morning

↓

About

Daylight

↓

Portfolio

Golden Hour

↓

Contact

Sunset

The progression should feel cinematic.

---

# Lighting Philosophy

Natural.

Soft.

Warm.

Human.

Editorial.

Never futuristic.

Never neon.

Never colorful.

---

# Background Lighting

Pure white remains dominant.

Lighting is created using extremely soft radial gradients.

Opacity should remain below visual awareness.

The background must never compete with photography.

---

# Documentary Lighting

Photography is the primary light source.

The interface should react to the photography,

not the opposite.

---

# Light Hierarchy

Photography

↓

Headline

↓

Evidence

↓

Interface

The user's eyes should naturally follow this order.

---

# Hero Transition

Portfolio inherits Hero's lighting language.

Only the temperature changes slightly.

The visitor should never notice a hard visual transition.

---

# Shadows

Very soft.

Large blur radius.

Low opacity.

Never dramatic.

Never floating card shadows.

Every shadow should suggest physical depth,

not UI depth.

---

# Highlight Strategy

Highlights should only appear on:

Documentary photography.

Primary project.

CTA.

Never highlight every component equally.

---

# Glass Effect

Avoid obvious glassmorphism.

If glass is used,

it should be almost invisible.

The visitor should perceive depth,

not transparency.

---

# Surface Treatment

Cards should feel like premium printed paper.

Not floating dashboards.

Not software panels.

Surface texture should remain calm.

---

# Evidence Lighting

Evidence annotations should feel attached to the photograph.

They inherit the photograph's lighting.

Never appear brighter than the image itself.

---

# CTA Lighting

The final CTA receives slightly more visual attention.

Not through color.

Through light.

Slightly brighter background.

Slightly cleaner spacing.

Slightly stronger contrast.

---

# Responsive Behaviour

Lighting intensity decreases on mobile.

Focus remains on readability.

Atmosphere is preserved without reducing performance.

---

# Performance Rules

Use CSS gradients.

Avoid expensive blur filters.

Avoid animated shadows.

Avoid multiple overlapping glow effects.

Maintain GPU-friendly rendering.

---

# Visual Checklist

✓ Calm atmosphere.

✓ Documentary feeling.

✓ Warm but subtle.

✓ Premium depth.

✓ Consistent with Hero.

✓ Photography remains dominant.

✓ No artificial lighting.

---

# Final Principle

Lighting should feel like sunlight entering a room.

Quiet.

Natural.

Almost invisible.

If visitors notice the photography instead of the lighting,

the lighting system has succeeded.
