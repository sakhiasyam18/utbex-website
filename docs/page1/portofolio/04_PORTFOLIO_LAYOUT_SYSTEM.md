# Introduction

Layout is storytelling.

Visitors should never feel they are scrolling through cards.

They should feel they are progressing through chapters.

---

# Layout Philosophy

Think like a documentary editor.

Not like a web designer.

Each block introduces new information.

Each block changes the visual rhythm.

Avoid repetition.

# Spacing

# Scroll Experience

The Portfolio must unfold gradually.

Each chapter should occupy enough vertical space to let visitors pause naturally.

Never compress multiple stories into one viewport.

The scroll rhythm should feel like walking through an exhibition.

Sequence

Observe

↓

Read

↓

Reflect

↓

Continue

Alternate visual density.

Heavy visual chapter.

↓

Light editorial chapter.

↓

Heavy visual chapter.

↓

Quiet whitespace.

This rhythm prevents visual fatigue.

---

# Final Principle

The visitor should never notice the layout.

They should only notice the story.

The layout succeeds when it quietly disappears.
