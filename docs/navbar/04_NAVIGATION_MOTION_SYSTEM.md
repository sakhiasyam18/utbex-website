It breathes.

# Initial Appearance

Navigation softly fades into the page.

Small translate.

Soft opacity.

No scaling.

No bounce.

No dramatic entrance.

---

# Scroll Behaviour

When scrolling down:

Navigation becomes slightly more solid.

Blur increases gently.

Shadow becomes a little deeper.

When scrolling upward:

Navigation softly returns to its lighter state.

Never snap.

Never jump.

---

# Hide Behaviour

If auto-hide is implemented:

Hide with small upward movement.

Fade slightly.

Never disappear instantly.

Immediately return when users scroll upward.

---

# Hover Interaction

Logo

Very small brightness increase.

Navigation Link

Thin underline grows.

Small opacity transition.

Tiny upward movement.

CTA

Very soft glow.

Small lift.

Maximum scale:

1.02

---

# Active State

Transition smoothly.

No flashing.

No strong color changes.

The active state should feel persistent,

not animated.

---

# Glass Animation

Glass should never pulse.

Blur never animates continuously.

Lighting remains stable.

Atmosphere should feel static,

not alive.

---

# Ambient Lighting

Allow extremely slow ambient movement.

Movement should take several seconds.

Almost impossible to notice.

The purpose is to prevent the interface from feeling flat.

---

# Mobile Drawer

Open

Fade.

Small horizontal slide.

Backdrop blur.

Close

Reverse exactly.

Maintain symmetry.

Never use spring bounce.

---

# Timing

Fast

Interaction feedback.

Medium

Hover.

Slow

Navigation reveal.

Very Slow

Ambient atmosphere.

---

# Reduced Motion

Respect

prefers-reduced-motion

Remove:

Ambient movement.

Floating effects.

Slide transitions.

Keep:

Opacity.

Instant state changes.

Accessibility always comes first.

---

# Performance

Use GPU transforms.

Transform.

Opacity.

Backdrop-filter.

Avoid expensive layout animations.

Avoid continuous repainting.
