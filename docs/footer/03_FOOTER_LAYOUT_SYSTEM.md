---

# Introduction

The Footer is the quietest composition on the website.

It should not introduce new visual complexity.

Instead,

it gently concludes the journey.

---

# Layout Philosophy

Think like the final page of a premium editorial magazine.

Large margins.

Few elements.

Clear hierarchy.

Comfortable breathing space.

Nothing feels crowded.

---

# Reading Flow

Closing Statement

↓

Supporting Description

↓

Navigation

↓

Contact

↓

Social

↓

Copyright

This order should never change.

---

# Closing Statement

This is the visual anchor.

Always positioned first.

Centered.

Comfortable reading width.

Never compete with other elements.

---

# Supporting Description

One short paragraph.

Maximum three lines.

Explain the invitation,

not the company.

---

# Navigation Block

Simple.

One horizontal row on desktop.

Vertical stack on mobile.

Maximum five links.

No dropdowns.

No secondary navigation.

---

# Contact Block

Keep only essential information.

Email.

Location.

Optional phone number.

Nothing else.

---

# Social Block

Small.

Balanced.

Placed naturally after contact.

Never dominate the layout.

---

# Copyright

Always the final element.

Small typography.

Quiet contrast.

Minimal spacing.

---

# Grid System

Desktop

12 Columns

Centered composition.

Tablet

8 Columns.

Balanced spacing.

Mobile

Single column.

Natural reading order.

---

# Alignment

Desktop

Center aligned.

Mobile

Center aligned.

Maintain one visual axis from top to bottom.

---

# Vertical Rhythm

Large spacing

↓

Medium spacing

↓

Small spacing

↓

Minimal ending

Create a gradual visual slowdown.

---

# Responsive Behaviour

Do not simply stack elements.

Preserve the editorial rhythm.

Visitors should experience the same emotional ending on every device.

---

# Layout Checklist

✓ Editorial hierarchy.

✓ Calm reading flow.

✓ Large breathing space.

✓ Clear navigation.

✓ Natural ending.

✓ Consistent with Portfolio.

---

# Final Principle

The Footer should never feel like another section.

It should feel like the final page of the same story,

allowing visitors to leave naturally and remember the experience.
