# Scope

Apply atmosphere globally.

Do not redesign sections.

Do not move layouts.

Do not replace typography.

Do not modify storytelling.

Only improve the invisible experience surrounding every section.

---

# Primary Goal

Transform multiple sections into one seamless experience.

Visitors should never feel that they are moving between separate pages.

Instead,

they should feel that every chapter exists within the same carefully crafted environment.

---

# Background System

Create one continuous background.

Never restart backgrounds between sections.

Allow the environment to flow naturally.

The canvas should feel endless.

---

# Ambient Lighting

Introduce extremely subtle ambient lighting.

Examples:

• oversized radial gradients

• warm white diffusion

• soft ivory lighting

• delicate maroon atmosphere

Opacity should remain extremely low.

Visitors should feel warmth,

not visual effects.

---

# Depth Layer

Create depth using:

Light.

Blur.

Spacing.

Layering.

Soft shadows.

Avoid decorative 3D objects.

Avoid exaggerated visual effects.

---

# Noise Texture

Introduce one global texture layer.

Opacity:

1–2%.

Monochromatic.

Soft.

Invisible.

Noise should support realism,

never attract attention.

---

# Section Transitions

Every transition should become softer.

Use:

Opacity.

Blur.

Spacing.

Ambient overlap.

Avoid decorative dividers.

Avoid hard section breaks.

---

# Global Rhythm

The website should breathe.

Some chapters become quieter.

Some become richer.

Never maintain identical pacing throughout the page.

Create emotional rhythm,

not repetitive rhythm.

---

# Motion

Atmosphere moves slowly.

Very slowly.

Allow:

Tiny ambient breathing.

Slow light diffusion.

Gentle opacity transitions.

Never create obvious animations.

---

# Performance

Prefer CSS.

Prefer gradients.

Prefer transforms.

Avoid videos.

Avoid canvas effects.

Avoid expensive rendering.

Beauty should remain lightweight.

---

# Accessibility

Atmosphere must never reduce readability.

Atmosphere must never reduce contrast.

Atmosphere must respect prefers-reduced-motion.

Accessibility always wins.

---

# Engineering Rules

Do not rename folders.

Do not restructure components.

Do not duplicate logic.

Modify only files that genuinely require atmospheric refinement.

Maintain production-ready quality.

---

# Final Validation

Before returning implementation verify:

✓ One continuous atmosphere.

✓ Smooth section transitions.

✓ Invisible ambient lighting.

✓ Premium depth.

✓ Consistent background.

✓ Editorial pacing.

✓ Lightweight rendering.

✓ Responsive behaviour.

✓ Accessibility preserved.

---

# Final Principle

You are not adding visual effects.

You are creating emotional continuity.

When visitors finish scrolling,

they should never remember where one section ended and the next began.

They should only remember one beautifully connected journey.
