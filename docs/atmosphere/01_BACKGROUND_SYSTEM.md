Audience:

Creative Director

---

# Introduction

The background is not empty space.

It is the environment in which every story unfolds.

A premium experience is never built on a flat white canvas.

Instead,

the background quietly creates emotion,

depth,

and continuity.

Visitors should never consciously notice the background.

They should simply feel that the website is alive.

---

# Philosophy

The background exists to support storytelling.

Never decorate.

Never compete.

Never distract.

Its purpose is to make every section feel naturally connected.

---

# Foundation

Every page begins with one continuous canvas.

Never create isolated backgrounds for each section.

Instead,

build one uninterrupted environment.

---

# Background Stack

The global background should be constructed in layers.

Layer 01

Pure White Canvas

↓

Layer 02

Soft Warm Gradient

↓

Layer 03

Large Ambient Light

↓

Layer 04

Blurred Diffusion

↓

Layer 05

Noise Texture

↓

Layer 06

Content

The visitor should never perceive individual layers.

Only one atmosphere.

---

# White Philosophy

White is the dominant color.

However,

white should never appear empty.

Use warm whites.

Soft whites.

Ivory whites.

Layered whites.

Avoid sterile white backgrounds.

---

# Ambient Gradient

Large gradients are encouraged.

Requirements:

• radial

• oversized

• blurred

• almost invisible

Never create obvious gradient circles.

The transition should disappear naturally.

---

# Global Background Movement

The background should not remain visually static.

Allowed:

very slow light drift

tiny opacity breathing

subtle ambient diffusion

Movement should require several seconds.

Visitors should barely notice it.

---

# Background Lighting

Large atmospheric lights may exist behind content.

Examples:

Hero

↓

Warm ivory radial light

About

↓

Neutral white diffusion

Portfolio

↓

Very soft balanced lighting

Impact

↓

Slightly brighter optimism

Footer

↓

Gradually fading light

The entire page should feel like one changing day,

not five different sections.

---

# Depth Philosophy

Depth belongs to the environment,

not components.

Use:

large blurred gradients

ambient diffusion

soft light fields

background layering

Avoid decorative 3D objects.

---

# Background Blur

Large invisible blur is encouraged.

Blur should remove sharp transitions between light sources.

Never blur content.

Blur only supports atmosphere.

---

# Edge Treatment

Avoid hard page edges.

Allow gradients to slowly disappear.

The page should feel endless.

---

# Decorative Elements

Avoid decorative illustrations.

Avoid random geometric shapes.

Avoid unnecessary patterns.

Everything should originate from light.

---

# Color Temperature

Primary

Warm White

Secondary

Soft Neutral

Accent

Very subtle UTBEX Maroon

Accent colors should remain extremely restrained.

---

# Noise Integration

Background noise should be global.

Never local.

Noise should unite every section.

Opacity should remain extremely low.

Its purpose is realism.

Not texture.

---

# Responsive Behaviour

Background atmosphere should scale naturally.

Never disappear.

Never become stronger on mobile.

The emotional experience remains identical.

---

# Performance

Prefer CSS gradients.

Prefer blur filters.

Avoid large image assets.

Atmosphere should cost almost nothing.

---

# Background Checklist

✓ Continuous.

✓ Warm.

✓ Invisible.

✓ Premium.

✓ Editorial.

✓ Lightweight.

✓ Emotional.

✓ Cohesive.

---

# Final Principle

The visitor should never notice the background.

They should only notice that the website feels calm,

alive,

and beautifully connected from the first section to the last.
