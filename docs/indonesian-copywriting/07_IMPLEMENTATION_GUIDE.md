# Primary Objective

Transform the existing website into a natural Indonesian experience.

Visitors should never feel they are reading translated English.

Every sentence should feel as though it was originally written by an Indonesian copywriter.

---

# Translation Philosophy

Never translate literally.

Always adapt naturally.

Preserve meaning.

Preserve emotion.

Preserve rhythm.

Do not preserve sentence structure.

Meaning matters more than grammar.

---

# Brand Voice

Every sentence should sound like UTBEX.

Human.

Calm.

Professional.

Optimistic.

Evidence-driven.

Never arrogant.

Never promotional.

Never bureaucratic.

---

# Storytelling Priority

Always communicate in this order.

People

↓

Journey

↓

Collaboration

↓

Impact

↓

Organization

UTBEX should never become the main character.

Communities always come first.

---

# Writing Rules

Prefer short paragraphs.

Prefer natural Indonesian.

Prefer active voice.

Avoid unnecessary jargon.

Avoid repetitive wording.

Avoid literal translation.

---

# Headline Rules

Headlines must be memorable.

Maximum:

Six words.

Easy to pronounce.

Easy to remember.

Avoid abstract language.

---

# CTA Rules

CTAs should invite.

Never pressure.

Preferred examples:

Mari Berdiskusi

Mulai Kolaborasi

Lihat Jejak Kolaborasi

Pelajari Cerita Kami

Avoid:

Hubungi Sekarang

Klik Di Sini

Daftar

Mulai Sekarang

---

# Navigation Rules

Navigation labels should remain short.

One or two words whenever possible.

Navigation should communicate value,

not structure.

---

# Portfolio Rules

Never describe projects.

Describe transformation.

Replace technical explanations with human outcomes whenever possible.

---

# Impact Rules

Never display statistics alone.

Every number requires context.

Every metric represents real people.

---

# Footer Rules

The Footer should conclude emotionally,

not informationally.

Visitors should leave with hope,

not simply contact information.

---

# Photography Relationship

Copy should never explain what photography already communicates.

Allow visuals to tell the emotional story.

Copy provides context,

not duplication.

---

# Motion Relationship

Animations support reading.

They never compete with text.

If motion distracts from copy,

reduce motion.

---

# Typography Relationship

Never increase text length simply to fill space.

Whitespace is part of the communication.

Short copy often creates stronger impact.

---

# Responsive Behaviour

Desktop.

Tablet.

Mobile.

The message never changes.

Only the layout adapts.

---

# Accessibility

Plain language.

Clear hierarchy.

Semantic headings.

Meaningful buttons.

Readable paragraphs.

Accessible communication is part of good storytelling.

---

# AI Generation Rules

When generating new copy,

always ask:

Would an Indonesian reader naturally say this?

If not,

rewrite it.

Then ask:

Does this sound like UTBEX?

If not,

rewrite again.

Repeat until both answers are yes.

---

# Validation Checklist

Before returning implementation verify:

✓ Indonesian-first.

✓ Natural wording.

✓ Brand Voice preserved.

✓ Emotional consistency.

✓ Documentary storytelling.

✓ CTA consistency.

✓ Navigation consistency.

✓ Accessibility preserved.

✓ Responsive copy.

✓ No literal translations.

---

# Engineering Rules

Do not rename components.

Do not modify architecture.

Do not change design unnecessarily.

Only update copy,

labels,

microcopy,

and supporting text.

Respect the existing Design System.

Respect the Atmosphere System.

Respect the Content Direction System.

---

# Final Principle

Do not think like a translator.

Think like the person who originally wrote the website.

When visitors finish reading,

they should believe that every sentence was written specifically for them,

not translated for them.

That is successful implementation.
