# Design Rules

Preserve one visual language.

Never introduce new styles.

Never redesign completed sections.

Never mix multiple design philosophies.

Maintain one premium editorial identity.

---

# Atmosphere Rules

The website should never feel flat.

Use atmosphere carefully.

Allowed:

• radial light

• ambient blur

• subtle reflections

• premium glass

• soft shadows

• quiet depth

Everything should remain elegant.

Never cyberpunk.

Never exaggerated.

---

# Motion Rules

Motion must always have purpose.

Small.

Calm.

Natural.

Never distracting.

Respect prefers-reduced-motion.

---

# Engineering Rules

Strict TypeScript.

Reusable architecture.

No duplicated logic.

Future CMS ready.

Production ready.

Maintain design consistency before adding features.

# Accessibility

Semantic HTML.

Keyboard navigation.

Meaningful labels.

Visible focus.

Accessibility is mandatory.
