---

# Introduction

Light is the invisible material of the UTBEX interface.

It creates atmosphere.

It creates depth.

It creates emotional warmth.

Shadows simply confirm that depth already exists.

---

# Philosophy

Never design with shadows first.

Design with light first.

Light defines the environment.

Shadow defines the object.

Together they create realism.

---

# Emotional Goal

Visitors should never consciously notice the lighting.

Instead they should feel:

Warm.

Calm.

Premium.

Refined.

Alive.

---

# Lighting Hierarchy

Primary Light

↓

Ambient Light

↓

Reflection

↓

Shadow

↓

Depth

Always follow this order.

---

# Primary Lighting

Large.

Soft.

Natural.

Invisible.

Primary lighting should simulate daylight entering a quiet studio.

Never create dramatic spotlights.

---

# Ambient Lighting

Allowed atmospheric lighting:

• radial white glow

• warm ivory diffusion

• subtle maroon ambience

• blurred light source

• gentle edge illumination

Opacity should remain below 5%.

---

# Background Atmosphere

Large sections may contain invisible light layers.

Examples:

• oversized blurred radial gradients

• soft floating light fields

• subtle background diffusion

These elements prevent the interface from feeling flat.

---

# Glass Lighting

Glass receives:

Top highlight.

Internal reflection.

Soft edge illumination.

Tiny ambient diffusion.

Never use obvious glossy reflections.

---

# Shadow Philosophy

Shadows should support light,

never dominate it.

Use:

Large blur.

Low opacity.

Wide spread.

Natural softness.

Avoid hard-edged shadows.

Avoid heavy black shadows.

---

# Floating Components

Floating elements should appear gently suspended.

Depth comes from:

Light.

Blur.

Shadow.

Spacing.

Never rely on shadow alone.

---

# Hero Atmosphere

Hero may contain the richest lighting.

Allowed:

Soft radial lights.

Background diffusion.

Blurred ambient glow.

Very subtle depth fields.

Everything should remain elegant.

---

# Navigation Atmosphere

Navigation may use:

Soft glass reflection.

Gentle ambient glow.

Warm edge lighting.

Invisible floating shadow.

Never appear luminous.

---

# Portfolio Atmosphere

Photography remains dominant.

Lighting only supports image presentation.

Never compete with photographs.

---

# Footer Atmosphere

The lighting gradually fades.

The experience becomes quieter.

The page naturally comes to rest.

---

# Three-Dimensional Depth

Depth should be created using:

Layering.

Light.

Blur.

Surface elevation.

Atmosphere.

Never use exaggerated perspective.

Never create fake 3D objects.

---

# Performance Rules

Use CSS gradients whenever possible.

Prefer blur over large image assets.

Avoid expensive rendering.

Keep effects lightweight.

---

# Responsive Behaviour

Lighting should scale naturally.

Never disappear completely.

Never become stronger on mobile.

Maintain emotional consistency.

---

# Lighting Checklist

✓ Soft.

✓ Warm.

✓ Premium.

✓ Invisible.

✓ Lightweight.

✓ Editorial.

✓ Atmospheric.

✓ Consistent.

---

# Final Principle

Visitors should never think,

"That lighting effect looks beautiful."

They should simply feel that the entire website feels alive.

That is the purpose of the UTBEX lighting system.
