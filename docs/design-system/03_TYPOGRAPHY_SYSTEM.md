---

# Introduction

Typography is the voice of the website.

It communicates before animation,

before imagery,

and before interaction.

Every type decision should strengthen clarity,

emotion,

and trust.

---

# Typography Philosophy

Typography should feel:

Editorial.

Premium.

Modern.

Quiet.

Confident.

Human.

The visitor should enjoy reading every section.

Never overwhelm.

Never perform.

---

# Reading Experience

The website should read like a premium magazine,

not a marketing landing page.

Large breathing spaces.

Comfortable line height.

Balanced paragraph width.

Readable hierarchy.

---

# Font Personality

Primary Typeface

Modern Sans Serif.

Highly legible.

Elegant.

Neutral.

Timeless.

Avoid decorative fonts.

Avoid futuristic fonts.

Avoid playful fonts.

---

# Heading Philosophy

Headings create emotion.

They should be:

Bold.

Confident.

Simple.

Easy to scan.

Never overly condensed.

Never exaggerated.

Large headings should become visual anchors.

---

# Body Philosophy

Body text explains.

Not decorates.

Keep paragraphs comfortable.

Maintain consistent rhythm.

Never use long unreadable text blocks.

---

# Reading Width

Ideal paragraph width:

45–70 characters.

Never stretch text across the screen.

Reading comfort is more important than filling space.

---

# Hierarchy

Display

↓

Heading

↓

Subheading

↓

Body

↓

Caption

↓

Micro Label

Never skip hierarchy levels.

---

# Font Weight

Use restraint.

Prefer:

Regular

Medium

SemiBold

Bold

Avoid excessive Black weights.

Avoid overly thin typography.

---

# Letter Spacing

Headings

Slightly tighter.

Body

Natural.

Labels

Slightly increased tracking.

Navigation

Comfortable spacing.

Never exaggerated.

---

# Line Height

Large Headings

Compact.

Body Text

Relaxed.

Long paragraphs

Generous.

Whitespace improves comprehension.

---

# Text Alignment

Prefer left alignment.

Center alignment only for:

Hero

Section Introductions

Footer Closing

Never justify text.

---

# Color Hierarchy

Primary

Near Black.

Secondary

Warm Gray.

Accent

UTBEX Maroon.

Avoid unnecessary color variation.

---

# Responsive Typography

Desktop

Large editorial scale.

Tablet

Slight reduction.

Mobile

Prioritize readability.

Never shrink typography excessively.

---

# Micro Typography

Pay attention to:

Quotation marks.

Dashes.

Spacing.

Punctuation.

Line breaks.

Every detail contributes to quality.

---

# Typography Checklist

✓ Readable.

✓ Calm.

✓ Elegant.

✓ Premium.

✓ Editorial.

✓ Consistent.

✓ Accessible.

✓ Human.

---

# Final Principle

Typography should never ask for attention.

It should quietly carry the story from beginning to end.
