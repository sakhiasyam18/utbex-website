# Introduction

Spacing is one of the strongest visual tools in the UTBEX Design System.

It creates rhythm.

It creates hierarchy.

It creates confidence.

Never think of spacing as empty space.

Spacing is part of the storytelling.

---

# Philosophy

The website should breathe.

Every section should have room to exist.

Visitors should never feel rushed.

Generous spacing creates premium perception.

---

# Visual Rhythm

Large

↓

Medium

↓

Small

↓

Micro

The rhythm should gradually slow the reading experience.

Avoid random spacing.

Everything follows a predictable system.

---

# Section Spacing

Between major chapters:

Very generous.

Hero

↓

About

↓

Portfolio

↓

Impact

↓

Footer

Each transition should feel intentional.

---

# Internal Spacing

Every section follows:

Headline

↓

Description

↓

Primary Content

↓

Supporting Content

↓

Ending

Maintain this rhythm consistently.

---

# Component Spacing

Components should never touch each other.

Always preserve breathing room.

Content groups should feel related,

but never crowded.

---

# Grid Rhythm

Spacing should follow a consistent scale.

Recommended progression:

4

8

12

16

24

32

48

64

96

128

Avoid arbitrary spacing values.

---

# Container Width

Wide enough for confidence.

Narrow enough for readability.

Never stretch content edge-to-edge.

Whitespace is part of the composition.

---

# Paragraph Spacing

Separate ideas naturally.

Avoid walls of text.

Create comfortable reading pauses.

---

# Card Spacing

Cards should feel independent.

Maintain generous padding.

Never compress content simply to reduce height.

---

# Navigation Spacing

Floating margins.

Balanced internal padding.

Comfortable click targets.

Navigation should feel weightless.

---

# Glass Components

Glass requires more breathing room than solid surfaces.

Allow shadows,

blur,

and ambient lighting to expand naturally.

Never clip visual effects.

---

# Responsive Behaviour

Desktop

Generous spacing.

Tablet

Slightly reduced.

Mobile

Compact,

but never cramped.

Maintain the same visual rhythm.

---

# Spacing Checklist

✓ Calm.

✓ Predictable.

✓ Premium.

✓ Comfortable.

✓ Consistent.

✓ Editorial.

✓ Human.

---

# Final Principle

Visitors should never consciously notice spacing.

They should simply feel that everything has room to breathe.
