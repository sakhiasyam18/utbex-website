# Atmosphere Philosophy

White remains the primary canvas.

However,

the website should never feel empty.

Invisible atmosphere is encouraged.

Allowed atmospheric elements include:

• soft radial lighting

• large ambient blur

• floating light gradients

• premium glass reflections

• subtle depth layers

• extremely soft glow

• elegant background diffusion

• quiet 3D visual depth

These elements exist only to create feeling,

never distraction.

---

# Three-Dimensional Language

Depth should come from light,

not decoration.

Use:

• soft floating surfaces

• layered compositions

• ambient shadows

• subtle elevation

• blurred light sources

Avoid:

• hard 3D objects

• exaggerated perspective

• gaming aesthetics

• futuristic cyberpunk

Everything should remain believable.
