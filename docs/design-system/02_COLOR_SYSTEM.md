---

# Introduction

Color is used to guide emotion,

not decorate the interface.

Every color must have a responsibility.

---

# Color Philosophy

The website should feel bright,

warm,

premium,

and timeless.

White remains dominant.

Accent colors create attention only when necessary.

---

# Primary Palette

Near Black

Accent

Everything else derives from these colors.

---

# Color Distribution

80%

White

Neutral backgrounds

20%

Text

Accent

Glass

Lighting

The interface should never feel colorful.

---

# White System

White is never flat.

Allowed:

• warm white

• ivory white

• subtle off-white

• layered white

Avoid cold blue whites.

---

# Accent Philosophy

Maroon represents identity.

Use it only for:

• CTA

• active navigation

• important keywords

• interaction feedback

Never use maroon as a large background.

---

# Neutral Colors

Neutral tones should support readability.

They create hierarchy,

not attention.

Prefer warm gray over cold gray.

---

# Glass Colors

Glass should inherit surrounding colors.

Recommended:

• translucent white

• soft neutral tint

• extremely subtle maroon reflection

Never use colored glass.

---

# Ambient Lighting

Background atmosphere may include:

• white radial light

• warm ivory glow

• soft maroon diffusion

• neutral light bloom

Opacity should remain extremely low.

Lighting should feel natural.

---

# Gradient Rules

Gradients should never become visible objects.

Only use:

• radial gradients

• large soft gradients

• atmospheric diffusion

Avoid:

• rainbow gradients

• strong linear gradients

• saturated backgrounds

---

# Shadow Colors

Avoid black shadows.

Prefer:

warm gray

neutral gray

soft maroon tint

Large blur.

Low opacity.

---

# Hover Colors

Hover should increase clarity,

not brightness.

Small contrast changes.

Small opacity changes.

Never flash.

Never oversaturate.

---

# Accessibility

Maintain WCAG AA minimum.

Text must always remain readable.

Never sacrifice accessibility for aesthetics.

---

# Color Checklist

✓ Warm.

✓ Premium.

✓ Human.

✓ Minimal.

✓ Atmospheric.

✓ Editorial.

✓ Accessible.

✓ Consistent.

---

# Final Principle

Color should quietly support the story.

Visitors should feel the atmosphere,

not notice the palette.
