# UTBEX DIGITAL EXPERIENCE SYSTEM

## PORTFOLIO MOTION SYSTEM

Version: 1.0

Status: LOCKED

Language: English

Audience:
Motion Designer
Frontend Engineer
AI Coding Assistant

---

# Introduction

Motion is invisible storytelling.

The visitor should never think,

"That animation looks beautiful."

Instead,

they should feel that every transition is natural.

Portfolio should feel slower than Hero.

More reflective.

More immersive.

---

# Motion Philosophy

Hero

Creates Attention

↓

About

Creates Emotion

↓

Portfolio

Creates Reflection

Motion should encourage visitors to pause.

Observe.

Think.

Continue.

---

# Core Principle

Every movement must reveal information.

Never decorate it.

If an animation does not strengthen storytelling,

remove it.

---

# Motion Hierarchy

Section Reveal

↓

Featured Story

↓

Project Cards

↓

Documentary Details

↓

Micro Interaction

Each layer has its own rhythm.

---

# Section Entrance

Portfolio should softly emerge from About.

No visible cut.

No dramatic fade.

Background evolves naturally.

Light becomes slightly warmer.

Spacing becomes more generous.

---

# Featured Story

Appears first.

Large image fades in.

Headline follows.

Story appears after.

Evidence appears last.

The visitor should never receive everything at once.

---

# Documentary Sequence

Photography

↓

Story

↓

Evidence

↓

Interaction

This order should never change.

---

# Card Reveal

Cards appear gradually.

Not simultaneously.

Create the feeling that new stories are being discovered.

Not loaded.

---

# Hover Philosophy

Hover should reward curiosity.

Not attract attention.

Image

Scale 1.02

↓

Shadow

Slightly stronger

↓

Caption

Slight emphasis

↓

Nothing else

---

# Evidence Animation

Evidence annotations should feel attached to the documentary.

Tiny fade.

Tiny translate.

Never floating.

Never bouncing.

Never animated continuously.

---

# Scroll Feeling

Scrolling should feel cinematic.

Like walking through an exhibition.

Not browsing a dashboard.

---

# Motion Timing

Fast

Reserved only for interaction feedback.

Medium

Typography.

Images.

Story.

Slow

Background.

Lighting.

Atmosphere.

---

# Reduced Motion

Respect

prefers-reduced-motion

Remove

Parallax

Continuous movement

Floating elements

Keep only

Opacity

Small translate

---

# Performance

Transform only.

Opacity only.

GPU accelerated.

Avoid expensive blur animations.

Avoid layout recalculation.

---

# Motion Consistency

Portfolio must inherit Hero motion language.

Never invent new animation styles.

Every movement belongs to the same design system.

---

# Micro Interaction

Micro interactions should reward curiosity.

Never demand attention.

Project Image

Soft scale.

Story Title

Very slight opacity transition.

Evidence

Gentle fade.

CTA

Small horizontal movement.

Cursor

Never exaggerated.

Never magnetic.

Never elastic.

Every interaction should feel effortless.

The visitor should notice the content,

not the animation.

---

# Motion Checklist

✓ Calm

✓ Documentary

✓ Elegant

✓ Invisible

✓ Consistent

✓ Lightweight

✓ Accessible

✓ Story Driven

---

# Final Principle

Motion should feel like someone quietly placing photographs onto a table.

Nothing should feel mechanical.

Nothing should feel digital.

Everything should feel human.
