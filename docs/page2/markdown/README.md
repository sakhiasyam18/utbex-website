# Overview

Page 2 bukan merupakan Company Profile.

Page ini adalah sebuah **digital experience** yang menceritakan perjalanan, karya, inovasi, dan dampak yang dibangun oleh **Arik Dwi Asmara** bersama UTBEX Indonesia.

Pengunjung tidak hanya melihat informasi, tetapi diajak mengikuti sebuah perjalanan bagaimana sebuah ide sederhana berkembang menjadi inovasi yang memberikan dampak nyata bagi masyarakat.

Website harus terasa seperti sebuah documentary experience, bukan website organisasi biasa.

---

# Main Objective

Membuat pengunjung percaya bahwa:

> "UTBEX bukan sekadar perusahaan."

tetapi

> "Sebuah ekosistem inovasi yang mampu mengubah ide menjadi dampak nyata."

dan

> "Mas Arik merupakan sosok di balik perjalanan tersebut."

# Development Principle

Semua keputusan desain harus selalu menjawab pertanyaan berikut:

Apakah section ini membuat pengunjung semakin percaya?

Apakah section ini memperkuat storytelling?

Apakah section ini memperlihatkan bukti nyata?

Apakah section ini memberikan pengalaman visual yang premium?

Jika jawabannya tidak,

section tersebut harus diperbaiki.
