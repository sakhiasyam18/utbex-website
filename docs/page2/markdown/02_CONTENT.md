# Content Philosophy

Page 2 bukan dibuat untuk menampilkan informasi sebanyak mungkin.

Page ini dibuat untuk membawa pengunjung mengikuti sebuah perjalanan.

Setiap section harus menjawab satu pertanyaan.

Setelah pertanyaan tersebut terjawab, pengunjung akan berpindah ke section berikutnya.

Website harus terasa seperti membaca sebuah documentary.

# Content Rules

Tidak menggunakan paragraf panjang.

Tidak menggunakan bahasa promosi.

Tidak menggunakan klaim berlebihan.

Lebih banyak visual.

Lebih sedikit teks.

Setiap section maksimal memiliki satu pesan utama.

# Closing Statement

Page 2 harus membuat pengunjung merasa telah mengenal perjalanan seorang inovator, memahami dampak yang telah dibangun, dan percaya bahwa mereka dapat menjadi bagian dari perjalanan berikutnya.
